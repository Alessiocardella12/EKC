This repository contains two folders: one containing the codes used for data analysis and another containing the data. Specifically, the analysis to visualize the Environmental Kuznets Curve for the BRICS countries was performed using only two specifications: a second-degree polynomial function and a third-degree polynomial function. Furthermore, the data for carbon dioxide come from EDGAR (Emissions Database for Global Atmospheric Research), while the data for GDP levels come from the World Bank. Each folder contains a brief descriptive explanation.
